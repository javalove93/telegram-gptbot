Available commands:

/help
  This help page

/clear
  GPT에 전달하는 대화 히스토리(컨텍스트)를 삭제하고 새로운 토픽으로 대화합니다
  Clear all the history(contexts) that have been told and start new topic

/topic
  현재 대화 중인 토픽을 표시합니다. 실제로는 GPT에게 "list up topics we are talking so far" 질문을 던집니다
  Display current topics. This is actually a question "list up topics we are talking so far" to GPT model

/params
  GPT 모델에 대한 하이퍼 파라미터 목록을 조회하거나 변경합니다. 추가적인 아규먼트가 없으면 다음과 같이 표시합니다

  (example) 
  frequency_penalty: 0.5
  presence_penalty: 0.5
  max_tokens: 2048
  temperature: 0.5

  다음과 같이 파라미터를 수정할 수 있습니다
  /params frequency_penalty  

  다음과 같이 파라미터를 디폴트 값으로 변경합니다
  /params reset

/system
  GPT 모델의 System role 메시지를 지정합니다. GPT 모델이 어떤 모드로 동작할지 지정합니다
  디폴트는 "You are a helpful assistant" 입니다
  예를 들어 한국어 챗봇처럼 답변하게 하고 싶으면 다음과 같이 입력합니다
  (example) "You're a korean chatbot that is answering in Korean"

/translate 
  Google Translator를 사용하여 질문(입력)을 영어로 번역하여 전달 후, 영어로 받은 응답을 한국어로 번경합니다
  Using Google Translator, the bot translate your question into English for GPT, and translate the answer from GPT into specified language

/history
  현재 대화중인 메시지(GPT의 입력 토큰)를 표시합니다
  Print messages(GPT input tokens)
